## Troubleshooting

**Extension does not detect any media**
Make sure the video is actively playing in the browser tab. Some pages only expose media sources after playback has started.

**Download fails or file is incomplete**
Check your internet connection and try again. If the issue persists, try refreshing the page and restarting playback.

**Popup does not appear**
Confirm the extension is installed and enabled. Try reloading the page and clicking the extension icon again.

**No quality options are shown**
Only variants detected during browser-network activity are listed. If only one source is found, that is the only option available.

**Download takes too long**
Larger files may take longer depending on your connection speed and the source server. Ensure you have enough storage space.
